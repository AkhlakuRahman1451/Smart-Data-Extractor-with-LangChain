# LangChain Data Extractor

A simple LangChain project that extracts structured information from unstructured text using an LLM.

## Features

- Load and connect an LLM
- Create custom prompts
- Build LangChain chains
- Extract structured data from text
- Retrieve:
  - First Name
  - Last Name
  - Country
- Support for both single and multiple inputs

## Technologies Used

- Python
- LangChain
- LLM

## Example Input

John Doe is from the United States.

## Example Output

{
  "first_name": "John",
  "last_name": "Doe",
  "country": "United States"
}

## Learning Objectives

This project was built to practice:

- Prompt Engineering
- LangChain Chains
- Structured Data Extraction
- LLM Integration
